# bullockcart_loadcheck
A simple Python program that checks if a bullock cart can carry given loads. If a load exceeds the cart capacity, it removes the load and continues checking the remaining loads.
# Bullock Cart Load Check

This project simulates a bullock cart carrying loads. The cart has a maximum weight capacity. If any load exceeds the capacity, it is removed and the next load is checked.

## 🚀 Features
- Accepts cart capacity
- Accepts multiple load weights
- Skips loads that exceed capacity
- Displays loads that can be carried

## 📥 Input
- Maximum capacity of the cart
- List of load weights (space-separated)

## 📤 Output
- Loads that are accepted
- Loads that are rejected
